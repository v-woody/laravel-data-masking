# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, report them privately by emailing jamie.woodruff@proton.me. You can also use GitHub's private vulnerability reporting feature on the Security tab of this repository.

Please include as much of the following as possible:

- A description of the vulnerability and its potential impact
- The steps to reproduce the issue
- The Laravel and PHP versions you are using
- Any relevant code or configuration

You will receive a response within 72 hours. Once the issue is confirmed, a fix will be prepared and a security advisory published before or alongside the release.

Please do not disclose the vulnerability publicly until a fix has been released.
